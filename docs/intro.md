# Introduction

Ensoul 是一款面向 DAO 的 SBT 发行管理工具，提供灵活的权限管理方案、细颗粒度的SBT 字段描述以及丰富的开发者接口，可用于组织成员的声誉，角色和权限等的管理。

Ensoul 由 SeeDAO 战略孵化。SeeDAO 为 Ensoul 启动提供初始的资金支持、内容和流量支持。
